## Responsive Behavior

### Mobile/Tablet View (< lg breakpoint)
- Filters move above results
- Sidebar widgets (Trending/Recommended) appear below filters
- Stack layout instead of columns
- Touch-friendly tap targets
- Collapsible sections help save vertical space

## Color & Style

All components use the design system:
- **Primary Blue** (#0284c7) for buttons, links
- **Gray Scale** for borders, backgrounds, text
- **Semantic Colors** for badges (success, warning, etc.)
- **Consistent Spacing** using design tokens
- **Card Component** for containers with shadows
- **Badge Component** for tags and filter chips
- **Button Component** for actions

## Accessibility

- All interactive elements are keyboard accessible
- Proper ARIA labels on buttons
- Focus indicators visible
- Semantic HTML structure
- Screen reader friendly

## Performance

- Debounced search suggestions (300ms)
- Lazy loading of event cards
- Pagination for large result sets
- localStorage for recent searches (client-side, no backend calls)
