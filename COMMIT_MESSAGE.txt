feat: Implement launch essentials (GDPR, DSA, Legal, SEO)

This commit implements critical features required for platform launch,
including GDPR compliance, DSA (Digital Services Act) compliance, legal
requirements, and SEO improvements.

## GDPR Compliance

### Data Export
- Add comprehensive data export functionality for user data portability
- Implement asynchronous background job processing for large exports
- Add DataExportSettings component with polling and download capabilities
- Create dataExportProcessor service with retry logic and expiry management
- Export includes: profile, events, comments, following/followers, attendance,
  likes, reports, and appeals
- Exports expire after 7 days and support up to 3 retry attempts
- Add exponential backoff hook for robust polling

### Privacy Settings
- Add public/private profile visibility controls (isPublicProfile field)
- Implement privacy helper functions for profile access checks
- Support follower-only private profiles with proper access control
- Update PrivacySettings component with new visibility options
- Add comprehensive privacy test coverage

## DSA (Digital Services Act) Compliance

### Reporting System
